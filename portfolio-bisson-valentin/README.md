# Portfolio — Valentin Bisson

Portfolio personnel de **Valentin Bisson**, étudiant en cybersécurité à **Ynov Rennes**.

Ce site présente mon profil, mes compétences, mes passions, mes projets publics GitHub et mes informations de contact.  
Il a été conçu comme une vitrine simple, moderne et responsive pour accompagner ma recherche d’alternance.

## Aperçu

Le portfolio contient plusieurs sections :

- **Accueil** : présentation rapide du profil.
- **À propos** : résumé de mon parcours et de mon objectif.
- **Passions** : sport automobile, photographie et jeux vidéo.
- **Compétences** : cybersécurité, réseau, développement web, Docker, Linux / Windows Server, Git / GitHub.
- **Projets publics** : affichage automatique des dépôts publics GitHub.
- **Contact** : email, LinkedIn et GitHub.

## Technologies utilisées

## Fonctionnalités

- Interface moderne avec thème sombre.

## Installation en local

1. Cloner le dépôt :

```bash
git clone https://github.com/nyekoTV/NOM-DU-DEPOT.git
```

2. Ouvrir le dossier :

```bash
cd NOM-DU-DEPOT
```

3. Lancer le site :

Il suffit d’ouvrir le fichier `index.html` dans un navigateur.

Ou avec l’extension **Live Server** sur Visual Studio Code.

## Configuration GitHub

Dans le fichier `script.js`, le nom d’utilisateur GitHub est défini ici :

```js
const GITHUB_USERNAME = "nyekoTV";
```

Si le compte GitHub change, il faut modifier cette valeur.

## Déploiement avec GitHub Pages

Pour mettre le portfolio en ligne :

1. Aller dans les paramètres du dépôt GitHub.
2. Ouvrir l’onglet **Pages**.
3. Dans **Build and deployment**, choisir :
   - Source : `Deploy from a branch`
   - Branch : `main`
   - Folder : `/root`
4. Enregistrer.

Le site sera ensuite disponible via une URL GitHub Pages.
